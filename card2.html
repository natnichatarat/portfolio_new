Mobile Application Chatbot

Developed a mobile application using Flutter for a dormitory recommendation chatbot. The system utilized Dialogflow for user interaction and conversational flow, and Firebase was implemented for both the database and user authentication.

Key Features:

  • Users can use the chatbot system
  • Owners can add dormitory information and other details of the dormitory
  • Users and owners must register and log in to their accounts
