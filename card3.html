Arduino Color Sorter

Develop an automatic color sorting simulation system using IoT technology and Arduino connected to a color sensor. Design and implement software for real-time data processing and visualization. The system efficiently detects colors by displaying a digital code for each color detected.

Key Features:

  • Can color sort
  • Can displaying a digital code(as 1 = red)
  • Can on-off servo on mobile
